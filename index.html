Random item generator for enemy drops

Health Regeneration
Gradually restores player health
-- Health Regen game.Players.PlayerAdded:Connect(function(player) player.CharacterAdded:Connect(function(character) local humanoid = character:WaitForChild("Humanoid") while true do wait(1) if humanoid.Health < humanoid.MaxHealth then humanoid.Health += 5 end end end) end)
Camera Shake Effect
Creates screen shake during explosions
-- Camera Shake local function shakeCamera(player, intensity, duration) local camera = workspace.CurrentCamera local originalPos = camera.CFrame for i = 1, duration do local offset = Vector3.new( math.random(-intensity, intensity), math.random(-intensity, intensity), math.random(-intensity, intensity) ) camera.CFrame = originalPos * CFrame.new(offset) wait(0.1) end camera.CFrame = originalPos end
